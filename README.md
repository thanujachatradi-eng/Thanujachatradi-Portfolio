# Thanujachatradi Portfolio Website

This repository contains my *personal project portfolio website*.  
It includes a *Login/Signup template* and a *Projects page* that showcases my mini-projects.  
The website is built using *HTML, CSS, and JavaScript*.

---

## 🚀 Features
- User-friendly *Login and Signup page*
- Responsive design using *HTML & CSS*
- Projects displayed in *card layout*
- Simple *JavaScript validation* for login
- Easy to customize and extend

---

## 🛠 Technologies Used
- HTML5
- CSS3
- JavaScript
